# 💰 جيبي (Geybi) - نظام الإدارة المالية الذكي

**جيبي** هو تطبيق ويب متكامل (Full-Stack) مصمم لمساعدتك على تتبع مصروفاتك، مداخيلك، ديونك، وجمعياتك المالية في مكان واحد، بواجهة عصرية وسهلة الاستخدام.

## ✨ المميزات الرئيسية
- **لوحة تحكم (Dashboard)**: ملخص سريع لحالتك المالية الحالية.
- **إدارة المصروفات والمدخولات**: تسجيل وتصنيف كل معاملة مالية.
- **مركز البطاقات**: تتبع بطاقات الائتمان ومواعيد كشوف الحساب.
- **إدارة القروض والديون**: متابعة المبالغ المتبقية ونسبة السداد.
- **الجمعيات (Money Circles)**: تنظيم الجمعيات المالية ومعرفة دورك في القبض.
- **السلف والتسليف**: تتبع المبالغ التي تدين بها للآخرين أو التي يدينون بها لك.
- **نظام تنبيهات ذكي**: تذكير بمواعيد الدفع والأقساط القادمة.
- **تصميم Dark Mode**: واجهة مريحة للعين وفخمة.

## 🛠️ التقنيات المستخدمة (Tech Stack)
- **Frontend**: React.js, Vite, TailwindCSS, Lucide Icons.
- **Backend**: Node.js, Express.js.
- **Database**: MongoDB (Atlas).
- **Authentication**: JWT (JSON Web Tokens), Bcrypt.js.

## 🚀 التشغيل المحلي (Local Setup)

### 1. المتطلبات
- Node.js مثبت على جهازك.
- حساب MongoDB Atlas أو MongoDB محلي.

### 2. إعداد السيرفر (Backend)
```bash
cd server
npm install
# قم بإنشاء ملف .env بناءً على .env.example واملأ البيانات
npm run dev
```

### 3. إعداد الواجهة (Frontend)
```bash
cd client
npm install
# قم بإنشاء ملف .env بناءً على .env.example واملأ الرابط
npm run dev
```

## 🌐 النشر (Deployment)
### ملاحظة هامة حول المجلدات الفرعية (Subdirectories)
تم إعداد هذا المشروع ليعمل تحت مسار فرعي باسم `/fin` (مثلاً: `mcprim.com/fin`). إذا كنت ستنشر المشروع على "النطاق الرئيسي" مباشرة (Root Domain)، يجب عليك:
1. إزالة `basename="/fin"` من ملف `client/src/App.jsx`.
2. إزالة `base: '/fin/'` من ملف `client/vite.config.js`.
3. تعديل `window.location.href = '/fin/login'` في ملف `client/src/api/axios.js`.

- **Backend**: يمكن نشره بسهولة على **Render** أو Heroku.
- **Frontend**: يمكن نشره على **mcprim.com** أو Vercel.
  - تأكد من بناء المشروع باستخدام `npm run build` ورفع مجلد `dist`.
